Site Design & Details

The site is hosted on Github, using Github Pages. Website code is statically generated, meaning that all content on this site was pre-generated and not rendered. There’s only one piece of Javascript used on this site, which is meant for expanding the menu for mobile users. I use Hugo, which is a static site generation framework written in Go. This allows me to write pages in Markdown, instead of needing to constantly format pages in HTML and grapple with CSS fun.

While I didn’t make Hugo, I did write the website theme “Huey” from scratch. I named it after the musician Baby Huey, whose record was on my mantle when I was trying to come up with a name. It’s MIT licensed, so feel free to use & modify accordingly. Huey uses Bulma CSS Framework, which saved on time and formatting issues.

RSS feed is automatically generated, so if you are interested in my musings please subscribe. The feed for posts is: https://adamwhitlock.com/archive/index.xml.
